# EVENT-MANAGEMENT-SYSTEM
EventCore is a responsive event management platform for colleges that connects students and organizers through dedicated portals, offering easy event planning and a modern, interactive user experience.

#Features

*Students can browse, register, and track events.

*Submit event proposals.

*Coordinators can create events and manage proposals.

*Task management using drag-and-drop Kanban board.

*Real-time analytics and reports.

*Google authentication for secure login.
